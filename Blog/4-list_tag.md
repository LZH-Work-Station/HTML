# 列表标签

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>列表学习</title>
</head>
<body>

<!--有序列表-->
<ol>
  <li>Java</li>
  <li>Python</li>
  <li>运维</li>
  <li>前端</li>
  <li>C</li>
</ol>

<!--无序列表-->
<ul>
  <li>Java</li>
  <li>Python</li>
  <li>运维</li>
  <li>前端</li>
  <li>C</li>
</ul>

<!--自定义列表-->
<dl>
  <dt>学科</dt>
  <dd>Java</dd>
  <dd>数学</dd>

  <dt>地方</dt>
  <dd>天津</dd>
  <dd>北京</dd>
</dl>

</body>
</html>
```

## 有序列表

- ol: 有序列表， 由1，2 ， 3， 4。。。排列的
- li: 列表中的内容， 一行一行的

## 无序列表

- ul：无序列表， 用黑点什么的来表示

## 自定义列表

- dl：自定义列表
- dt：列表的名称，title
- dd：列表的内容
